# Accessibility Checklist

Use this checklist to make the app easier to use for more people.


## Keyboard

- [ ] User can navigate important pages using keyboard
- [ ] Focus state is visible
- [ ] Buttons and links can be activated with keyboard
- [ ] No keyboard trap exists

## Forms

- [ ] Inputs have visible labels
- [ ] Required fields are clear
- [ ] Error messages explain what went wrong
- [ ] Error messages are close to the related field

## Text And Contrast

- [ ] Text is readable on desktop
- [ ] Text is readable on mobile
- [ ] Color contrast is strong enough
- [ ] Text does not overlap other elements

## Images And Icons

- [ ] Product images have useful alt text
- [ ] Decorative images do not confuse screen readers
- [ ] Icon-only buttons have accessible labels

## Layout

- [ ] Page works at mobile width
- [ ] Page works at desktop width
- [ ] Zooming does not break the layout
- [ ] Important content is not hidden off-screen
